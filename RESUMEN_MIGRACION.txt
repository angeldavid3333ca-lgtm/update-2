🚀 CÓMO EMPEZAR
═════════════════════════════════════════════════════════════════════════════

PASO 1: Verificar la instalación
────────────────────────────────────
Ejecuta en terminal:
  $ php verificar_instalacion.php

PASO 2: Inicializar la base de datos
─────────────────────────────────────
OPCIÓN A - Desde web:
  1. Abre: http://localhost/proyasist/setup_db.html
  2. Haz clic en "Inicializar Base de Datos"

OPCIÓN B - Desde terminal:
  $ php db_init.php

PASO 3: Acceder al sistema
──────────────────────────
  1. Abre: http://localhost/proyasist/index.html
  2. Selecciona tu tipo de usuario
  3. Usa las credenciales de datos_ejemplo.txt

🎯 CAMBIOS TÉCNICOS PRINCIPALES
═════════════════════════════════════════════════════════════════════════════

1. CONEXIÓN:
   ❌ ANTES: $pdo = new PDO("mysql:host=localhost;...", $user, $pass);
   ✅ AHORA: require_once 'db_config.php';

2. DUPLICADOS:
   ❌ ANTES: ON DUPLICATE KEY UPDATE estado=?, observaciones=?
   ✅ AHORA: INSERT OR REPLACE INTO tabla (campos) VALUES (?)

3. CONFIGURACIÓN:
   ✅ Centralizada en db_config.php
   ✅ PDO automáticamente disponible como $pdo
   ✅ Claves foráneas habilitadas en SQLite

✨ VENTAJAS DE SQLITE
═════════════════════════════════════════════════════════════════════════════

✓ No requiere servidor externo
✓ Base de datos embebida en un archivo (asistencia.db)
✓ Ideal para desarrollo local
✓ Fácil de respaldar (copiar el archivo)
✓ Compatible con PHP PDO
✓ Bajo consumo de recursos
✓ Transacciones ACID
✓ Múltiples índices

═════════════════════════════════════════════════════════════════════════════

⚠️ NOTAS IMPORTANTES
═════════════════════════════════════════════════════════════════════════════

1. SQLite está optimizado para desarrollo local
2. Para producción con alta concurrencia, considera MySQL
3. El archivo asistencia.db debe ser escribible por el servidor web
4. Realiza copias de seguridad regularmente
5. Las contraseñas aquí son SOLO para desarrollo
