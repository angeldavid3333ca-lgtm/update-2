🚀 FLUJO DE USO
═══════════════════════════════════════════════════════════════════════════════

1. INICIALIZAR BD
   ├─ Opción A: Ejecuta setup_db.html en navegador
   ├─ Opción B: php db_init.php en terminal
   └─ Verifica con: php verificar_instalacion.php

2. ACCEDER AL SISTEMA
   ├─ URL: http://localhost/proyasist/index.html
   ├─ Selecciona tipo de usuario
   └─ Usa credenciales de datos_ejemplo.txt

3. USAR EL SISTEMA
   ├─ Alumno: Ver asistencia propia
   ├─ Docente: Registrar asistencia
   ├─ Padre: Consultar asistencia de hijo
   └─ Admin: Gestionar usuarios y reportes

SQLite vs MySQL:

Característica             SQLite          MySQL
─────────────────────────────────────────────────
Servidor externo          ✓ No            ✗ Sí
Configuración              ✓ Mínima        ✗ Compleja
Instalación               ✓ Fácil         ✗ Difícil
Respaldar                 ✓ Copiar archivo ✗ Export
Portabilidad              ✓ Excelente      ✗ Media
Desarrollo local          ✓ Perfecto       ✓ Excesivo
Producción local          ✓ Bueno         ✓ Mejor
Alta concurrencia         ✗ Limitada      ✓ Excelente
